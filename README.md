# write_like_trump
 ml5.js + tensorflow + trumptwitterarchive.com

A [ml5js](https://ml5js.org/) application (built on top of [tensorflow.js](https://js.tensorflow.org/)) that uses a pretrained model on the corpus of tweets by Donald Trump, and outputs written text in his style.

* Model: [LTSM](https://en.wikipedia.org/wiki/Long_short-term_memory) built with a [Colaboratory Notebook](https://colab.research.google.com/drive/1vG_cfKEGzdwX64wrBbf9egq2gWijk8Id) with GPU environment.  Code was based on [ml5-data-and-training tools](https://github.com/ml5js/ml5-data-and-training/tree/master/training/lstm)
* Dataset: [Trump Tweets](trump_tweets.csv) spanning May 4th 2009 to June 9th 2018   via [trumptwitterarchive.com](http://www.trumptwitterarchive.com/archive) ( converted to text with a [Colaboratory Notebook](https://colab.research.google.com/drive/1Q6SCSNnoe7wri92O4qWLSD79sSaCN2_3) )


Inspired by [/Currie32/Tweet-Like-Trump](https://github.com/Currie32/Tweet-Like-Trump)
