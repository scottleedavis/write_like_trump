# write_like_trump
tensorflow + ml5.js + trumptwitterarchive.com

A [ml5js](https://ml5js.org/) application (built on top of [tensorflow.js](https://js.tensorflow.org/)) that uses a pretrained model on the corpus of tweets by Donald Trump, and outputs written text in his style.

* Model: built with [ml5-data-and-training tools](https://github.com/ml5js/ml5-data-and-training/tree/master/training/lstm)
* Dataset: Trump Tweets spanning May 4th 2009 to June 9th 2018   via [trumptwitterarchive.com](http://www.trumptwitterarchive.com/archive) ( [Colab conversion](https://colab.research.google.com/drive/1Q6SCSNnoe7wri92O4qWLSD79sSaCN2_3)
