# tweet_like_trump
tensorflow + tensorflow.js + trumptwitterarchive.com

A [tensorflow.js](https://js.tensorflow.org/) application that uses a pretrained model to output tweets typed, in the style of Donald Trump.  Builds on the work of [/Currie32/Tweet-Like-Trump](https://github.com/Currie32/Tweet-Like-Trump)

Parts
* [Model Colab page](https://colab.research.google.com/drive/1Q5Qh1d4wuDQlgp1LfuUr6AtFVT3TcfOz)
* Dataset: Trump Tweets spanning May4th 2009 to June 9th 2018   via [trumptwitterarchive.com](http://www.trumptwitterarchive.com/archive)
